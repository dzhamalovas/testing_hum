# Memory Solver

Это утилита-решатель для задач в стиле memory matching (по скриншоту поля).

## Как использовать

1. Загрузите скриншот.
2. Если на скриншоте есть лишние области (панели/фон), выделите мышкой только область с карточками.
3. Укажите `rows` и `cols`.
4. Нажмите «Найти пару».

Программа сравнивает все карточки внутри выбранной области и подсвечивает 2 наиболее похожие.

## Запуск

```bash
python3 -m http.server 8000
```

Откройте <http://localhost:8000>.
