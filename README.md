# Memory Solver

Это не игра, а утилита-решатель для задач в стиле memory matching.

## Что делает

- принимает скриншот поля с карточками;
- делит поле на сетку `rows x cols`;
- сравнивает все карточки между собой по пикселям;
- выделяет 2 наиболее похожие (обычно это одинаковые картинки).

## Запуск

```bash
python3 -m http.server 8000
```

Откройте <http://localhost:8000>, загрузите изображение и нажмите «Найти пару».
